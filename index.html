Projects

SmashBuzz LINK

Developed in React using a serverless AWS Lambda backend in NodeJS, and MongoDB. This app leverages data gathered from Twitter’s API and Smash.gg’s GraphQL endpoint to display recent and upcoming tournaments with related mentions on Twitter.

Magic Mirror Module

Developed and deployed on a Raspberry Pi. Utilizing javascript within the ‘Magic Mirror’ framework, this module queries recently posted games from the itch.io website (via an RSS feed) and displays them in a cycling feed on the screen.

Open source contributions

I've submitted fixes on GitHub for bugs I've discovered in a few tools that I use. Includes software built on NodeJS, .NET, and others.

3D Level Design Link

I designed and modeled the physical layout for this stage with 3D modeling tools. An artist gave it some style and it's now included the official release for Warsow, an open source first person shooter.

One Step from Eden Mods

I collaborated with a small team (Eden's Treasures) to create two mods for One Step from Eden using C# and LUA. Published on Steam Workshop with 500+ downloads. The two mods are "Bolts and Volts" and "Juggler's Pack", which add new core mechanics and a number of new assets for additional gameplay and replayability.

Fractured Rogue

Fractured Rogue is a Roguelike game developed on the virtual Pico-8 system in LUA.

Castlevania Aria of Sorrow: Reprise Link

Reprise is a RomHack of Castlevania Aria of Sorrow that changes the core gameplay to become a completely randomized, endless roguelike. Tons of customized content and gameplay modifications written in over 5000 lines of assembly.

Renew Arena

This FPS game was made in 7 days for the 7DFPS game jam as well as ProcJam (2020). It was built in Unity using C# and features procedurally generated levels.

Minetest Mod - Cube Mobs

This mod for Minetest in LUA that provides a custom set of enemies. 1200+ downloads on ContentDB.
